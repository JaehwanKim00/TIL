# 작성한 코드
```sql
SELECT 
        I.ITEM_ID, I.ITEM_NAME

FROM 
        ITEM_INFO AS I

JOIN 
        ITEM_TREE AS T 

ON
        T.ITEM_ID = I.ITEM_ID

WHERE 
        T.PARENT_ITEM_ID IS NULL

ORDER BY
        I.ITEM_ID
```
